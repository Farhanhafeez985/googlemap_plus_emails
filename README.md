# 🚀 Project Overview

This project automates lead generation using Google Maps.

It consists of two components:

## 1. Google Maps Scraper
- Extracts business data from Google Maps
- Saves results into CSV format

## 2. Email Scraper (Optional)
- Reads CSV generated by Script 1
- Visits business websites
- Extracts additional email addresses

> ⚠️ Note: The first script already extracts emails using a browser extension. The second script is optional for better email coverage.

# 📦 Installation Flow (IMPORTANT ORDER)

Follow this exact order:

---

## Step 1: Install Python 3.10

### Windows
Download:
https://www.python.org/ftp/python/3.10.0/python-3.10.0-amd64.exe

✔ Enable: Add Python to PATH

Verify:
```bash
python --version
```

---

### macOS
Download:
https://www.python.org/ftp/python/3.10.0/python-3.10.0post2-macos11.pkg

Verify:
```bash
python3 --version
```

---

### Linux
```bash
sudo apt update
sudo apt install software-properties-common
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt install python3.10 python3-pip
```

---

## Step 2: Create Virtual Environment

```bash
pip3 install virtualenv
cd <project-folder>
virtualenv myenv --python=python3.10
```

---

## Step 3: Activate Virtual Environment

### macOS / Linux
```bash
source myenv/bin/activate
```

### Windows
```bash
myenv\Scripts\activate
```

---

## Step 4: Install Dependencies

```bash
pip3 install scrapy
pip3 install openpyxl==3.1.5
pip3 install playwright==1.58.0
pip3 install pyexcel==0.7.4
pip3 install pyexcel-io==0.6.7
pip3 install scrapy-xlsx==0.1.1
```

---

## Step 5: Install Playwright Browsers

### Full install (recommended)
```bash
playwright install
```

### OR lightweight (Chromium only)
```bash
playwright install chromium
```
